# GoScan
**Raw Packet Syn Scanner Library**
```
$ GoScan --help

Usage of GoScan:
  -r string
        IP/CIDR Range (Required)
  -c int
        Requests per second (default 500)
  -p string
        Comma separated ports (default "22, 53, 66, 80, 81, 445, 457, 1080, 1100, 1241, 1352, 1433, 1434, 1521, 1944, 2301, 3128, 3306, 4000, 4001, 4002, 4100, 5000, 5432, 5800, 5801, 5802, 6346, 6347, 7001, 7002, 8080, 8888, 30821")
  -t int
        Timeout after request is sent (default 10)
```
